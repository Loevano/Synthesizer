LFO Core

Output Spread

This first LFO implementation modulates the summed oscillator output level per channel. Phase spread and unlinked mode are therefore channel-aware immediately. Routing the LFO to pitch or filters should happen later via a modulation matrix.

Voices

Each voice has its own active state, base frequency, gain, and output matrix. Routing is additive, so a voice can send to more than one output channel at the same time. Manual routing changes switch the synth routing preset to Custom.

Oscillators

Oscillator controls are scoped to the selected voice. Each slot can be enabled, gain staged, and given its own waveform.
